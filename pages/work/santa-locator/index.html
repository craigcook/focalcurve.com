Santa Locator

December, 2022

{% endblock %} {% block article_body %}

This was a simple little landing page for a fun social media thing promoting Mozilla VPN around the 2022 holidays. The “Earthwide Location Framework” is, of course, imaginary. A little progress bar animates with a semi-randomized delay before seeming to time out and display an error message.

The little cartoon red panda (dubbed Panda Clause) was initially delivered as a GIF but that turned out to be a pretty huge file. Efforts to optimize the file size reduced the image quality to an unacceptable level. In the end a CSS sprite animation was the most performant way to bring this cozy fella to life.
